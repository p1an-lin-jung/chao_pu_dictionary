
## 陈恩泉《潮普词典》OCR+LLM整理版

大部分潮汕方言资料均为“字典”，这本是少有的词典，收录了非常多词汇和口语句子，目前流传的电子版为图片版，无法检索、复制。本仓库使用最新的paddle-OCR-VL对每一页进行字符识别，并用Claude模型对识别后的结果进行整理。原字典页面中存在的无意义标注和冗余信息被过滤。

## 潮普词典部分标注说明

在潮普词典中，潮汕话的词条用“【】”包裹，普通话的词条用“〖〗”包裹。

潮普词典的拼音方案是作者在“粤教潮拼”、“国际音标”的基础上，改造过的方案（下文称为“潮普注音”），之所以用自创的方案，应该是方便作者编辑、输入。声调的标注采用0到9，并且如果有变调，标注的是变调后的调类。


### 注音方案
词典中的拼音是按照 **“揭阳话”** 为准。
